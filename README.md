# BPLD - Bangladeshi-Plant-Leave-Dataset
A Hierarchical Image Dataset of Bangladeshi Fruit, Flower, Medicine Plant Leave

## Overview

**BPLD (Bangladeshi Plant Leaves Dataset)** is a self-collected hierarchical plant leaf image dataset developed for research in deep learning, computer vision, and plant species recognition.

The dataset contains original full-resolution leaf images captured under real-world outdoor conditions using a personal mobile phone camera. No resizing or synthetic generation has been applied to the images in this version.

The dataset follows a two-level hierarchical structure:

- **Level 1 (Main Classes):** Flower, Fruit, Medicinal  
- **Level 2 (Subclasses):** 6 subclasses under each main class  

---

## Dataset Statistics

- **Total Images:** 7,200  
- **Image Type:** Original full-resolution images  
- **Main Classes:** 3  
- **Total Subclasses:** 18  
- **Images per Subclass:** 400  
- **Images per Main Class:** 2,400  
- **Class Distribution:** Fully balanced  

---

## Class Structure

### Flower (2,400 images)
Hibiscus, Lotus, Marigold, Rose, Rose Periwinkle, Togor  

### Fruit (2,400 images)
Guava, Jackfruit, Lemon, Mango, Papaya, Wood Apple  

### Medicinal (2,400 images)
Bitter Vine, Neem, Pathorkuci, Thankuni, Tulshi, Vasaka  

---

## Image Collection

All images were captured by the dataset creator using a personal mobile phone under natural environmental conditions. The dataset preserves real-world variability in:

- Lighting conditions  
- Background complexity  
- Leaf orientation  
- Scale and perspective  

No synthetic or AI-generated images were used.
